> Персонален уебсайт, разработен с фокус върху бързина, модерен UI и техническа елегантност. Създаден от [Станчев SEO | Оптимизация за търсачки](https://stanchev.bg)

## Технологии

Този проект използва съвременен стек за разработка на уеб приложения:

-  **[Vite](https://vitejs.dev/)** – ултра-бърз билд инструмент
-  **[Tailwind CSS](https://tailwindcss.com/)** – утилитарна CSS рамка
-  **[TypeScript](https://www.typescriptlang.org/)** – типобезопасен JavaScript
-  **[Netlify](https://www.netlify.com/)** – хостинг и CI/CD

---

##  Структура на проекта

```bash
├── public/               # Статични файлове
├── src/                  # Изходен код
│   ├── assets/           # Изображения, икони и др.
│   ├── components/       # Повторно използваеми компоненти
│   ├── pages/            # Основни страници на сайта
│   └── main.ts           # Входна точка на приложението
├── index.html            # Главният HTML файл
├── tailwind.config.js    # Tailwind конфигурация
├── vite.config.ts        # Vite конфигурация
└── package.json          # Зависимости и скриптове
```

---

##  Скриптове

```bash
# Инсталиране на зависимости
npm install

# Стартиране на проекта в development режим
npm run dev

# Билд за продукция
npm run build

# Преглед на production билд локално
npm run preview
```
